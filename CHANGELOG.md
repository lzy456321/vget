## [2.0.0] — 2024 · 五项重大升级

### ✨ 新功能

- **🪞 国内镜像下载** — yt-dlp 核心组件现在优先从国内镜像拉取，自动轮换三个备用源，再也不用对着转圈圈发呆
- **⏯ 断点续传** — 网断了？没事，重新跑一次接着下，不从零开始
- **🔔 下载完成通知** — Termux 用户下载完自动弹系统通知，手机可以扔一边去做别的事
- **🍪 自动读取浏览器 Cookie** — 自动尝试读取 Chrome / Firefox / Edge 的登录状态，B站大会员、YouTube 会员内容不再是拦路虎
- **📜 下载历史记录** — 每次下载自动存档，用 `--history` 随时回看，用 `--clear-history` 一键清空

### 🔧 改进

- yt-dlp 下载失败时错误提示更清晰，直接告诉你去哪手动下
- Cookie 读取失败时不再报错退出，改为友好提示继续运行
- 版本号升级至 v2.0.0

### 🐛 修复

- 修复部分 Termux 环境下 `bin/` 目录权限问题
- 修复播放列表模式下文件名冲突问题

## [1.1.0] — 2024 · 手机相册同步

### ✨ 新功能

- **📱 自动同步到手机相册** — Android 用户下载完视频自动复制到 `/sdcard/Movies/vget/`，打开相册直接看，不用去文件夹里翻

### 🔧 改进

- 下载完成后增加文件列表展示，清晰显示文件名和大小

## [1.0.0] — 2024 · 初次发布

### 🎉 首次发布

这是 vget 的第一个公开版本。从零开始，一个人在手机上用 Termux 折腾出来的。

### ✨ 功能清单

- **零安装** — 只需 `bash vget.sh 链接`，首次运行自动下载核心组件
- **全平台支持** — YouTube · B站 · Twitter · TikTok · Instagram · Twitch · 1000+ 网站
- **多系统兼容** — Linux · macOS · Android Termux · Windows WSL
- **仅音频模式** — `--audio` 一键提取 MP3
- **画质选择** — `--quality 720/1080/best`
- **播放列表下载** — `--list` 整个列表一起下
- **字幕下载** — `--subtitle` 中文优先
- **代理支持** — `--proxy` 解决地区限制
- **ffmpeg 自动检测** — 有 ffmpeg 自动启用最高画质合并

**彩蛋 #1 — 关于这个项目的诞生**

vget 是在一个普通的早上，用一部 OPPO 手机、一个 Termux 终端、和一个 AI 的对话里诞生的。

没有电脑，没有 IDE，没有任何"正经"的开发环境。

Token 泄露过一次（已作废），git 装了两次，`.gitignore` 因为文件名有点号找不到，上传文件时 README 消失了又回来了。

但最后它跑起来了。视频下载成功了。相册里出现了那个视频文件。

这就够了。
